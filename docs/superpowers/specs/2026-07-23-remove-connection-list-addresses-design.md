# 移除左侧连接列表地址信息设计

## 目标

精简左侧数据库连接列表：不再展示数据库地址和 SSH 地址，同时保留 SSH 隧道图标，便于识别连接方式。

## 范围

修改 `SortableConnectionItem` 的连接摘要区域：

- 删除数据库地址及数据库名文本：`host:port / database`。
- 删除 SSH 地址文本：`SSH: host:port`。
- 保留数据库类型图标、连接名称、SSH 隧道图标、连接状态标签、拖拽入口和操作按钮。
- 不修改连接配置、连接行为、数据库树或其他页面中的地址信息。

## 实现与验证

删除两段次级文本节点，并移除不再使用的相关包装节点。扩充 `ConnectionListGroups` 组件测试，使用带 SSH 配置的连接断言：连接名称和 SSH 隧道图标仍存在，数据库与 SSH 的地址文字均不在 DOM 中。最后运行该测试文件和前端构建。

## 自查

- 未留待定项或模糊展示规则。
- 改动仅影响左侧连接列表的展示，不改变连接数据或交互。
- 测试覆盖保留图标与移除两类地址文字这两个验收条件。
