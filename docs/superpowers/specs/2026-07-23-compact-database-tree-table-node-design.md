# 左侧数据表节点单行化设计

## 目标

简化左侧数据库树中的数据表节点：移除预估行数和收藏入口，让表/视图图标与表名处于同一行；过长表名保持单行并以省略号截断。

## 范围

仅调整 `src/components/database/DatabaseTree.tsx` 中的表节点标题渲染及其组件测试，不修改表元数据获取、收藏数据或其他列表。

## 设计

- 表节点标题只渲染表名，不再渲染 `rows` 值、"行"文本或收藏星标。
- 保留 Ant Design Tree 的 `icon` 配置：普通表使用绿色表图标，视图使用黄色视图图标。标题仅含表名后，Tree 的图标与标题自然位于同一行。
- 表名使用可收缩的单行文本容器，并通过 `ellipsis` 与现有树标题的 `min-width: 0` 规则隐藏溢出内容；完整名称继续由悬浮提示提供。
- 保留表注释的悬浮提示行为，不影响点击表节点、排序、加载和右键操作。

## 验收与测试

- 数据表节点不再展示行数或收藏按钮。
- 表/视图图标与表名显示在单行。
- 长表名不换行，溢出时可截断，并保留完整名称提示。
- 更新 `DatabaseTree` 组件测试，覆盖移除行数与收藏入口的行为及单行截断样式。
