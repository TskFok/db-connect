# 表详情页签栏标题设计

## 目标

在表详情页「数据」「结构」等内层页签所在行的最右侧，显示当前选中表的完整标识：`数据库/schema.表名`。

## 方案

- 复用 `TableContent` 已读取的 `selectedDatabase` 与 `selectedTable`，不发起新的数据请求。
- 在内层 `Tabs` 的同一水平行右侧渲染只读标题，名称格式固定为 `${selectedDatabase}.${selectedTable}`。
- 标题占用剩余空间并右对齐；名称过长时以省略号截断，并通过原生 `title` 属性保留完整值。
- 表或数据库/schema 切换时，标题随 Zustand 选中状态同步更新。

## 不包含

- 不改变顶部多表标签、当前内层页签、数据加载或表结构加载逻辑。
- 不恢复已移除的内容区数据库/表类型表头。

## 验收

- 选中 `mydb.users` 时，页签栏右侧显示 `mydb.users`。
- 切换至另一张表后，标题更新为新表标识。
- 现有「数据」「结构」等页签仍可正常显示与切换。
