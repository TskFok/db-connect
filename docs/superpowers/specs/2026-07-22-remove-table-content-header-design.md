# 移除数据表内容区表头设计

## 目标

移除数据表页面内容区顶部用于展示“数据库 / 数据表 / TABLE（或 VIEW）”的整行表头，使下方的数据、结构等标签页自然上移。

## 范围

- 删除 `TableContent` 内独立的表头信息节点及其 12px 下边距。
- 清理仅由该表头使用的 Ant Design 排版组件和图标导入。
- 保留顶部多表标签栏、标签页内容和底部状态栏中的数据库/表信息。
- 不改变表与视图的标签页能力、切换逻辑或数据加载逻辑。

## 实现方案

直接移除表头 JSX，让 `Tabs` 成为 `TableContent` 容器中的首个可见子元素。相比 CSS 隐藏或增加显示开关，这种方案不会保留无效布局，也不会引入当前不需要的配置状态。

## 验证

- 增加组件测试，确认内容区不再渲染数据库名、表名和 `TABLE`/`VIEW` 类型标签。
- 保留并验证数据、结构等标签页仍可见。
- 运行 `TableContent` 相关测试、前端测试集以及构建检查。
